# AI Resume Analyzer

An intelligent web application that analyzes resumes and provides insights to help candidates improve their chances of getting shortlisted. The system extracts information from resumes and compares it with job requirements to identify missing skills, strengths, and improvement areas.

---

##  Features

- Upload and analyze resumes
- Extract important information from resumes
- Compare resume skills with job description requirements
- Suggest missing skills and improvements
- User-friendly web interface
- Secure backend using Spring Boot

##  How It Works

1. User uploads a resume.
2. The backend processes the resume.
3. Skills and keywords are extracted.
4. The system compares them with job requirements.
5. Suggestions and improvements are generated.
